# Numeric Calculator

This is a simple calculator program built in C++.

I made this project to practice basic C++ syntax, arithmetic operations, and user interaction. It helped me apply beginner programming concepts in a straightforward way while getting more comfortable with C++.

## Concepts Practiced

- C++
- Arithmetic operations
- User input
- Conditional logic
- Basic program structure

## What I Learned

This project helped me strengthen my understanding of how to take user input, process it, and return a useful result.
